Generaptor
==========

**Generaptor** is a generator for Velociraptor-based offline collectors.
It allows you to quickly create pre-configured Velociraptor collectors with custom
rules, profiles, and targets for incident response and forensic investigations.

Project Overview
----------------

Generaptor is designed to simplify the creation of Velociraptor collectors by:

- Creating ready-to-use collectors with minimal configuration
- Supporting multiple platforms (Windows, Linux, and macOS)
- Enabling customization of collection rules
- Organizing collection rules by profile and targets to cover different use cases
